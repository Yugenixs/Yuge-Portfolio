# Ilyas Tio Afrilian/Yuge — Portfolio

Personal portfolio website showcasing my journey as a Data Science student.

Built with React, Vite, and Tailwind CSS. Features full-screen video backgrounds, glassmorphism UI, and smooth scroll navigation.

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS
- Lucide React

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## Deployment

Deployed on [Vercel](https://vercel.com). Any push to `main` branch will trigger an automatic redeployment.
